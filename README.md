作品名：ドイツ語単語マッチング〜未知のドイツ語単語との出会いにときめこう〜

作品説明：ドイツ語の単語学習を補助するアプリ。表示される単語の意味を答えて、正誤を○△✕で入力する。出題順には定着度合いが加味されているのがポイント。嫌いな単語とはたくさん出会えるよ(^^)
